<h1>Aoba</h1>
A screen capture/upload client like puush but with game capture support.

<h3>Currently Implemented</h3>
<ul>
<li>Single Screen Capture</li>
<li>All Screen Capture</li>
<li>Screen Region Capture</li>
<li>File Uploads</li>
</ul>
